## 1\. 실습용, 프로젝트용 파일 구분

### 실습용

* `app.py` 터미널에서 실행
  
    ```
    OSP_team11
    ├─ app.py
    ├─ database.py
    ├─ static
    │  ├─css
    │  │      style.css
    │  ├─images
    │  └─js
    │          main.js
    ├─ templates
    │      index.html
    │      list.htmll
    │      qna.html
    │      reg_items.html
    │      reg_reviews.html
    │      result.html
    │      review.html
    │      user_login.html
    └─     user_signup.html
  ```
